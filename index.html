Bootstrap Confirm Button

A plugin for showing in-place confirmation before proceed.

Download Github Repo

Demo

Click this button to see the result

Dependencies

  1. jQuery
  2. Bootstrap 3

Usage

Include the dependencies

Include Bootstrap style, jQuery, and bootstrap-confirm-button.js

  1. <link href="//netdna.bootstrapcdn.com/bootstrap/3.0.3/css/bootstrap.min.css" rel="stylesheet">
  2. <script type='text/javascript' src="//ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>
  3. <script type='text/javascript' src="bootstrap-confirm-button.js"></script>

Insert HTML

Insert a button into your page. Remember to use bcb-button for your button class name and bcb-label for the label

<button class="btn btn-warning bcb-button"><i class="glyphicon glyphicon-trash"></i> <span class="bcb-label">Delete</span></button>

Call bootstrapConfirmButton(options)

                $('.bcb-button').click(function (event) {
                    //this function will be called when user confirmed.
                });
                $('.bcb-button').bootstrapConfirmButton();

Options

class
The class to be added to the button on the first click. Default to btn-danger
message
Message to be display on bcb-label.
